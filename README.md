# Treap-Implementation


Treap is a data structure that combines the features of Tree and Heap data structure. Each node in the treap has a search key that complies with the feature of the binary search tree where the key value of a node must be larger than that of its left child and smaller than that of its right child. Also, each node possesses a random priority in the treap that complies with the property of a min-heap.
